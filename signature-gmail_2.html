How to install

Gmail (web)

  1. Open this file in a browser. Click into the white box you want, select all of it (click at the start, drag to the end), and copy.
  2. In Gmail, go to Settings (gear) → See all settings → General → Signature.
  3. Create a new signature, paste into the editor, and save changes at the bottom.
  4. Set it as the default for new emails and replies in the dropdowns just below.

Apple Mail (macOS)

  1. Open this file in Safari. Select all of the signature inside the white box and copy.
  2. In Mail, go to Mail → Settings → Signatures. Pick the account, click the plus to add one, and paste.
  3. Uncheck Always match my default message font so the styling and headshot are kept.

Headshots show as clean squares so they render the same in Gmail and Apple Mail (Gmail drops rounded corners). Email clients fall back to Georgia and Arial since they do not load the brand web fonts; colors, logo, and layout carry the brand. Logos, headshots, and social links pull from the live site and CDN, so they stay current.
