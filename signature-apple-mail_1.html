How to install in Apple Mail (macOS)

  1. Open this file in Safari.
  2. Click into the white box for the person, select all of it (click at the start, drag to the end), and copy.
  3. In Mail, go to Mail → Settings → Signatures. Pick the account, click the plus to add a signature, select all of the placeholder text it inserts, and paste over it.
  4. Uncheck Always match my default message font so the styling and photo are kept.
  5. Send yourself a test email to confirm the photo shows.

If the photo still does not paste, let the page fully load in Safari first (give the images a second), then select and copy. Because the images are embedded, there is nothing to fetch from a server. Email clients fall back to Georgia and Arial since they do not load the brand web fonts; colors, logo, and layout carry the brand.
